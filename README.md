# FastAPI Native AI Streaming (Lite) ⚡

A lightweight demonstration of token-by-token LLM streaming bypassing heavy frameworks like LangChain.

### Why this?
Heavy wrappers add latency and make custom tool-routing a nightmare. This repo demonstrates the raw WebSocket connection for instant, sub-100ms streaming natively in FastAPI.
